# ✉️ Cover Letter Generator

Paste a job description, upload your resume, and get a tailored human-sounding cover letter in seconds.

## Live App
[Coming soon]

## What it does
- Upload your resume as a PDF
- Paste any job description
- Choose a tone: Professional, Confident, or Conversational
- Get a cover letter under 250 words that sounds like a human wrote it

## Built with
- Python, LangChain, Groq, Streamlit
